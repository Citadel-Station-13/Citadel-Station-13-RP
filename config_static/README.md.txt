# Jukebox Music

Jukebox music goes here.

Tracks should be in JSON's with the organization format of `genre/artist.json`, replacing 'genre' and 'artist' with the genre and artist.

## Formats

All music should be `.mp3` tracks upon further addition.

## Allowed Hosts

All links **must** be `https://`.

Please use well-reputable hosts for security reasons.

We do not currently enforce a host whitelist but reserve the right to add one.
